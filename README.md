# ecoPDF

Annotateur PDF frugal et local.

ecoPDF permet d'ouvrir un PDF, d'écrire dessus et de télécharger la version annotée.

Aucune donnée n'est envoyée sur internet.

Fonctionne directement dans le navigateur.

## Fonctionnalités

- ouvrir un PDF
- écrire au doigt ou stylet ou à la souris
- annuler
- déplacer / scroller
- exporter le PDF annoté

## Philosophie

ecoPDF s'inscrit dans une approche de **sobriété numérique**.

- pas de cloud
- pas de compte
- pas de publicité
- pas de dépendance GAFAM

## Utilisations

- remplir un document administratif
- annoter un dossier
- accompagner la médiation numérique
- usage éducatif ou social
